# Assignment 6: 머꼬로 작성한 range
학번: 202255601

이름: 전준

## 1. 문제

두 개의 정수를 입력받아, 두 수 사이의 모든 정수를 포함하는 리스트를 생성하여 출력하는 프로그램을 작성한다.  
이때 시작값과 끝값도 리스트에 포함된다. 만약 첫 번째 정수가 두 번째 정수보다 크다면 공 리스트 `()`를 출력해야 한다.

예:
- 입력: 3, 7 → 출력: (3 4 5 6 7)  
- 입력: 5, 4 → 출력: ()  

---

## 2. 전략

이 문제는 재귀를 이용한 리스트 생성 문제이다.

기본 아이디어:
1. 시작값 > 끝값 → 공 리스트 반환
2. 아니면:
   - 현재 값을 리스트에 추가
   - 다음 값으로 재귀 호출

즉,
range(a, b) = (a . range(a+1, b))

머꼬에서는 `짝`을 이용하여 리스트를 구성한다.

---

## 3. 언어의 기능

- 재귀 (람다, 정의)
- 리스트 생성 (짝)
- 조건문 (조건)
- 비교 연산 (>)

머꼬는 리스트 기반 언어이므로 리스트를 직접 구성하는 방식이 핵심이다.

---

## 4. 구현 코드

```scheme
(정의 시작 (읽기))
(정의 끝 (읽기))

(정의 범위
  (람다 (시작 끝)
    (조건
      ((> 시작 끝) '())
      (#참 (짝 시작 (범위 (+ 시작 1) 끝))))))

(쓰기 (범위 시작 끝))
```
## 5. 토의

같은 기능을 구현하는 다른 방법으로는 꼬리 재귀를 사용할 수 있다.  
현재 구현은 `짝`을 이용해 현재 값을 앞에 붙이고, 나머지 범위를 재귀적으로 생성하는 방식이다.

다른 방법으로는 보조 함수를 만들어 결과 리스트를 누적하면서 처리할 수 있다.

```scheme
(정의 범위보조
  (람다 (현재 끝 결과)
    (조건
      ((> 현재 끝) 결과)
      (#참 (범위보조 (+ 현재 1) 끝 (짝 현재 결과))))))

(정의 범위
  (람다 (시작 끝)
    (범위보조 시작 끝 '())))
```

다만 이 방식은 `짝`으로 앞에 계속 붙이기 때문에 결과가 역순으로 만들어질 수 있다.  
따라서 올바른 순서로 출력하려면 리스트를 다시 뒤집는 과정이 필요하다.

## 6. 결론

이번 과제를 통해 머꼬에서 리스트를 생성하고 처리하는 방법을 익힐 수 있었다.  
특히 `짝`을 이용하면 현재 값을 리스트의 앞에 붙일 수 있고, 재귀를 이용하면 반복문 없이도 연속된 정수 리스트를 만들 수 있다는 점을 알게 되었다.

가장 중요하게 배운 점은 함수형 언어에서는 반복문보다 재귀를 중심으로 문제를 해결한다는 것이다.  
처음에는 재귀 호출의 흐름이 헷갈렸지만, 시작값이 끝값보다 커지는 순간 공 리스트를 반환하고, 그 앞에 값들이 차례대로 붙으면서 리스트가 완성된다는 구조를 이해할 수 있었다.
