# Assignment 6: 머꼬로 작성한 range
학번: 202255601

이름: 전준

## 1. 문제

두 개의 정수를 입력받아, 두 수 사이의 모든 정수를 포함하는 리스트를 생성하여 출력하는 프로그램을 작성한다.  
이때 시작값과 끝값도 리스트에 포함된다. 만약 첫 번째 정수가 두 번째 정수보다 크다면 공 리스트 `()`를 출력해야 한다.

예:
- 입력: 3, 7 → 출력: (3 4 5 6 7)  
- 입력: 5, 4 → 출력: ()  

---

## 2. 전략

이 문제는 재귀를 이용한 리스트 생성 문제이다.

기본 아이디어:
1. 시작값 > 끝값 → 공 리스트 반환
2. 아니면:
   - 현재 값을 리스트에 추가
   - 다음 값으로 재귀 호출

즉,
range(a, b) = (a . range(a+1, b))

머꼬에서는 `짝`을 이용하여 리스트를 구성한다.

---

## 3. 언어의 기능

이 프로그램을 구현하는 데 가장 도움이 된 언어 기능은 `짝`과 재귀이다.

머꼬에서 `짝`은 하나의 원소와 리스트를 받아 새 리스트를 만드는 기능이다.  
따라서 현재 값을 리스트 앞에 붙이면서 범위 리스트를 구성할 수 있다.

또한 `람다`와 `정의`를 이용하여 `범위` 함수를 만들고, 함수 내부에서 자기 자신을 다시 호출하여 반복문 없이 연속된 정수를 처리하였다.

조건문 `조건`은 재귀를 끝내는 기준을 정하는 데 사용되었다.  
`시작` 값이 `끝` 값보다 커지면 더 이상 리스트를 만들지 않고 공 리스트 `'()`를 반환한다.

---

## 4. 구현 코드

```scheme
(정의 범위
  (람다 (시작 끝)
    (조건
      ((> 시작 끝) '())
      (#참 (짝 시작 (범위 (+ 시작 1) 끝))))))
```
## 5. 토의

같은 기능을 구현할 때 꼬리 재귀를 사용할 수도 있다.  
꼬리 재귀는 함수의 마지막 동작이 자기 자신을 다시 호출하는 형태이다.  
이 방식은 재귀 호출 이후에 추가 계산이 남지 않기 때문에 반복문과 비슷한 구조로 처리할 수 있다.

```scheme
(정의 범위보조
  (람다 (현재 끝 결과)
    (조건
      ((> 현재 끝) 결과)
      (#참 (범위보조 (+ 현재 1) 끝 (짝 현재 결과))))))

(정의 범위
  (람다 (시작 끝)
    (범위보조 시작 끝 '())))
```

위 코드에서 `범위보조`는 현재 값, 끝 값, 누적 결과를 인수로 받는다.  
`현재`가 `끝`보다 커지면 누적된 `결과`를 반환하고, 그렇지 않으면 현재 값을 결과에 붙인 뒤 다음 값으로 재귀 호출한다.

이 방식은 마지막에 실행되는 연산이 `범위보조` 호출이므로 꼬리 재귀 구조라고 볼 수 있다.  
다만 `짝`은 리스트의 앞에 값을 붙이기 때문에 결과 리스트가 역순으로 만들어질 수 있다.

## 6. 결론

이번 과제를 통해 머꼬에서 리스트를 생성하고 처리하는 방법을 익힐 수 있었다.  
특히 `짝`을 이용하면 현재 값을 리스트의 앞에 붙일 수 있고, 재귀를 이용하면 반복문 없이도 연속된 정수 리스트를 만들 수 있다는 점을 알게 되었다.

가장 중요하게 배운 점은 함수형 언어에서는 반복문보다 재귀를 중심으로 문제를 해결한다는 것이다.  
처음에는 재귀 호출의 흐름이 헷갈렸지만, 시작값이 끝값보다 커지는 순간 공 리스트를 반환하고, 그 앞에 값들이 차례대로 붙으면서 리스트가 완성된다는 구조를 이해할 수 있었다.
